## 簡報說明

才多多介紹

在今年也協助工研院 外國碩、博士學曆人才服務，在台適應課程

這是在上週 台大校區舉辦的活動

目前平台求職類型， 對於企業有效人才來說有三大類，分別為 僑外生、藍領移工、新住民

目前還是以僑外生比例最多

 平台求職國籍分佈，以菲律賓及印尼居多，越南為次，語言上也是以英文為主居多

平台年齡分佈 以 20-29 到39人數最多

在學歷來說 還是以大學學士居多

這些服務是與在台 服務僑外生 多年合法公司合作

明年度針對畢業僑外生 在提供居留許可時，原本只能找工作不能工作，但目前有機會開放 2年個人工作許可 對象-在台畢業僑外生或者世界前200大畢業學生
