# 2026年 加薪換移工名額 計算公式

## 簡單說明

```
移工名額 = 本國勞工加薪人數 × 1人
（最多 = 投保人數 × 10%）
```

---

## 企業試算範例

### 範例公司：50人企業

| 項目 | 數字 |
|------|------|
| 勞保投保人數 | 50人 |
| 10%上限 | 50 × 10% = 5人 |
| 最多可取得移工名額 | 5人 |

### 如何計算？

1. **先算出10%上限**
   - 投保人數 × 10% = 可增加移工數

2. **再看你要加薪多少本國勞工**
   - 加薪1人 = 可多1位移工
   - 但不能超過10%上限

### 實際試算

| 投保人數 | 10%上限 | 可增加的移工名額 |
|----------|---------|------------------|
| 10人 | 1人 | 最多1人 |
| 30人 | 3人 | 最多3人 |
| 50人 | 5人 | 最多5人 |
| 100人 | 10人 | 最多10人 |
| 200人 | 20人 | 最多20人 |

---

## 重要提醒

1. **加薪對象**：月投保薪資最低的全時本國勞工
2. **加薪金額**：每人月薪 ≧ 2,000元
3. **每名移工**：仍須繳納每月2,000元就業安定費
4. **總移工上限**：不得超過員工總數45%

---

## 試算公式

```
最大移工名額 = 勞保投保人數 × 10%
（但不得超過總員工數的45%）
```
