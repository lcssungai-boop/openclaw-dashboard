# 勞動基準法（勞基法）

**資料來源：** 全國法規資料庫  
**最新修訂：** 民國114年

---

## 重點條文摘要

###  Worker Rights 基本權利
- 每週工時40小時
- 加班費計算
- 特休假計算
- 資遣費規定

### 與移工相關
- 適用對象：全體勞工（含外國人）
- 最低工資保障
- 工時規範

- 加班費計算方式
