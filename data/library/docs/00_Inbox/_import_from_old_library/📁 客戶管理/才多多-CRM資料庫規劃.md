## 潛在客戶分級（Lead Scoring）

### BANT 評估標準
| 維度 | 評估項目 |
|------|----------|
| **B**udget 預算 | 客戶是否有明確預算？ |
| **A**uthority 決策權 | 聯絡人是否有決策權？ |
| **N**eed 需求 | 需求是否明確？ |
| **T**iming 時機 | 何時需要完成？ |

### ABC 分級標準

**A級（高優先）**
• 4項全符合：預算有、決策者、需求明確、時間緊迫
• 主動安排簽約，設定行動截止點
• **跟進頻率**：每日或隔日

**B級（有潛力）**
• 符合2-3項：需要進一步確認
• 用「下一步」語言引導進度
• **跟進頻率**：每週2-3次

**C級（需培養）**
• 符合0-1項：需求不明確或時間未確定
• 創造需求，安排免費諮詢
• **跟進頻率**：每月1-2次
