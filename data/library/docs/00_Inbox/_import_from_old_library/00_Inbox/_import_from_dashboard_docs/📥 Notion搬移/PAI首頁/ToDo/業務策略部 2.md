以下再做確認，函下來後何時要用完？以及注意事項：

**1. 調薪證明與申報期限**

一旦您獲得「加薪增額移工」的**招募許可函**，您必須確保在**引進或聘僱該名額移工之前（或最遲於聘僱許可申請時）**，已經完成對本國勞工的實質加薪及申報。

- **同步調整勞保級距：** 您應在加薪事實發生之日起，依法向勞保局申報調整該名本國勞工的投保薪資等級（至少調高一級）。
- **定期查核制度：** 勞動部會透過勞保投保資料進行**自動勾稽**與**定期查核**。若查獲加薪事實不符或事後減薪，將會以「1比1」的比例廢止對應的移工聘僱許可。

**2. 許可函效期（引進期限）**

- **六個月內完成入境：** 根據《外國人聘僱許可及管理辦法》，雇主在取得許可通知之日起 **6 個月內**，必須完成所聘僱外國人的入境手續或接續聘僱手續，逾期則許可撤銷。
- **加薪持續性：** 若該名移工 3 年聘僱期滿需再續聘，雇主屆時必須**再次**為本國基層全時勞工加薪 2000 元，才能繼續保有該額度。

**3. 注意事項**

- **對象限制：** 加薪對象必須是「全職」本國勞工，部分工時勞工不列入計算。
- **薪資範圍：** 2000 元的加薪必須屬於「經常性薪資」（如底薪、固定津貼、全勤獎金等），不包含加班費。

如果您已取得許可函但尚未完成勞保等級調整，請務必於移工到職前向 勞動部勞工保險局 完成申報，以避免後續查核不通過導致許可被廢止。您可以登入 [**勞動部勞動力發展署線上申辦系統**](https://fw.wda.gov.tw/) 追蹤您的案件進度與查核狀態。
