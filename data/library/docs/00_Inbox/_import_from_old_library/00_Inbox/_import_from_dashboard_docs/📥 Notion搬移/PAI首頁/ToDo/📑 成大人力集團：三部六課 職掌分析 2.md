# 🏗️ 扁平化管理：三部六課體系

## 1. 業務部 (前台) - 營收引擎

- 開發課：市場拓展、新客戶開發、需求媒合及維護客戶。責任：達到季/年度新簽約目標或移工人數。
- 行銷課：品牌經營、口碑行銷、KOC合作。責任：獲取穩定流量與品牌曝光。

## 2. 客服部 (中台) - 品質保證

- 翻譯課：即時溝通、合約解析、生活輔導。責任：確保移工與雇主間溝通零障礙。
- 宿管課：宿舍管理、生活關懷、緊急事件處理。責任：提升移工居住滿意度與合規性。

## 3. 行政部 (後台) - 穩定後盾

- 文件課：政府申辦流程、合約歸檔、法律遵循。責任：100% 準點提交政府各項申請。
- 財務部：薪資計算、稅務管理、收支控管。責任：財務數據準確性與金流優化。

<aside>
💰 **📌 核心薪酬策略：**
業務獎金 20% 應與「行銷轉化率」緊密掛鉤。行銷課負責引入流量，開發課負責轉化成交，雙方共享營收成長果實，確保公司利潤最大化。

</aside>

## 💰 20% 獎金分配架構定義

為了確保公司成長與服務品質並重，將 20% 獎金池細分為「開發獎勵」與「維護獎金」兩大類別：

- **🔥 獎金 (獎勵性質)：**針對「從無到有」的貢獻。包含開發全新客戶、成功推動公司新專案、或現有客戶申請「全新案件」之首期獎勵。此類旨在鼓勵業務積極開拓。
- **🛡️ 維護 (團隊性質)：**針對長期穩定經營的貢獻。包含既有合約的持續服務、定期規費收繳、行政作業流程維護。發放形式傾向於年終獎金、季獎金、或團隊績效獎金，旨在鼓勵客服與行政團隊的精實營運。
