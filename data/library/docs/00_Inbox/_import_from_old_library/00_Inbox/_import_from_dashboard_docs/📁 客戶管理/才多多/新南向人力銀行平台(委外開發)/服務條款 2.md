# 服務條款

會員不得在無正當理由下，將帳號與密碼 **轉讓、出租、分享或借予他人使用**。

若帳號遭他人盜用，請立即聯繫   網站名稱   客服並依指示處理。

因個人疏失或不當使用帳號所造成之後果，應由會員自行負責。

  網站名稱   依不同會員身分提供不同服務，並保留 **隨時調整服務內容之權利**。

服務內容包含但不限於：

- 招募服務
- 求職服務
- 廣告服務
- 會員中心與其他會員相關服務

### **系統維護與中斷**

為維持最佳服務品質，網站可能因系統異常、設備維護、障礙排除或通訊公司服務中斷而暫停服務，暫停期間會員權益不受影響。

### **付費服務說明（企業會員）**

- 企業使用招募或其他付費服務，即視為同意購買相關服務
- 網站名稱   可因版面或設計需求調整廣告顏色、排版與字型
- **公司 Logo 與廣告內容不會被更動**

電子發票將透過第三方平台或財政部電子發票整合系統寄送至會員指定 Email。

### **違規與退費**

- 若會員違反相關法律或網站規範，將被停止使用付費服務
- 已支付之費用 **不予退還**
- 企業一旦正式啟用付費服務，亦不得要求退費
- 若未於期限內使用完畢服務，仍視為已使用完畢

  網站名稱   有權隨時修改本服務條款，修改後內容將公告於網站。

若您不同意修改內容，請停止使用網站並通知我們停止處理您的個人資料。

如本條款有未盡事宜，將依台灣相關法律、習慣、誠信原則與對等原則處理。

若發生訴訟，雙方同意以 **台灣台南地方法院** 為第一審管轄法院。
