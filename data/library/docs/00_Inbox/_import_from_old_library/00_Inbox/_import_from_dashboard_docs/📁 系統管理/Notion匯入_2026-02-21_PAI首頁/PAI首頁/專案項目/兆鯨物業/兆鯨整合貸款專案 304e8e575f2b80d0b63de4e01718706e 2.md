# 兆鯨整合貸款專案

### 1. 策略一：信保基金「大額度中長期週轉金」整合案

目前您的永豐貸款雖有信保，但期限過短（24-30期）。

- **具體作法**：利用公司 401 報表的營收實力，向往來最優質的銀行（如玉山或三信）申請「中小企業信用保證貸款-營運週轉金」。
- **目標**：將還款期數爭取拉長至 **7 年（84期）**。
- **預期效益**：將目前中租與永豐約 1,000 萬的「超短債」併入此案。月付款將從原本的 **57 萬** 瞬間降至約 **13~15 萬**，立刻釋放每月 40 萬的現金流。

### 2. 策略二：申請「寬限期（Grace Period）」轉換

若短期內轉貸手續不及辦理，可針對現有銀行房貸進行協商。

- **具體作法**：向中國信託、高雄三信等房貸行申請「償還本金寬限期」，通常可爭取 1~2 年。
- **預期效益**：在寬限期內「只付利息、不還本金」。以您 1.3 億的房貸規模計算，若有一半能申請到寬限期，每月可減少 **30~40 萬** 的現金支出，作為緩衝中租高額還款的銀彈。

### 3. 策略三：中租保證金抵充（解約止血）

根據您的談話備忘，中租案（兆鯨）存有 **200 萬保證金**。

- **具體作法**：這筆錢在租賃期滿前是死錢。若能透過銀行增貸取得低利資金後，立即進行「提早解約」，並要求以 200 萬保證金直接扣抵尾款。
- **預期效益**：除了停止高達 9%~12% 的利息失血，更能直接註銷這筆月付 33.5 萬的負擔。
