# 月休8天法律陷阱分析

**來源：** 陳業鑫律師（前任法官、勞動局長）
**日期：** 2026年2月6日
**存放位置：** sungAI共用/法規遵循

---

## 📌 核心問題

「月休8天」在數學與法律上，有很高機率已經處於違法邊緣。

---

## 🔑 七個關鍵點

### 1. 法律沒有「月休」概念

《勞動基準法》第36條規定：
- 每7日應有2日之休息
- 1天例假（絕對不能出勤）
- 1天休息日（可加班）

**計算：**
- 一年52週 × 2天 = 104天（例假+休息日）

---

### 2. 國定假日16天

《勞基法》第37條：
- 一年應休總天數 = 104天 + 16天 = **120天**

**每月平均：**
- 120天 ÷ 12個月 = **10天/月**

**問題：**
- 「月休8天」= 每月少給2天假
- 一年少24天假

---

### 3. 四週變形工時不是免死金牌

餐飲業適用的「四週變形工時」：
- 只能調整休假分布
- 不能減少休假總數

**計算：**
- 13個四週循環 × 8天 = 104天
- + 16天國定假日 = 120天

**違法：**
- 月休8天 = 一年96天假
- 120 - 96 = **短少24天**

---

### 4. 如何合法化？關鍵在「錢」

**「月休8天」合法的條件：**
- 短少的2天視為「休息日加班」
- 依加班費率發給工資（前2小時1.33倍、後2小時1.67倍）

**勞動契約應明確約定：**
1. 本職缺採月休8天制
2. 短少之休假日折算加班費
3. 薪資明細分列「本薪」與「加班費」
4. 不能用「責任制」「統包」帶過

---

### 5. 2026年最低工資檢核

**2026年基本工資：**
- 月薪：29,500元
- 時薪：196元

**案例計算：**
- 月薪30,000元（含全勤、加班費）
- 扣除加班費約2,000-3,000元
- 本薪可能只剩28,000元

**違法：** 本薪低於29,500元最低工資
**罰則：** 2萬-100萬元，可加重至150萬元

---

### 6. 最可怕風險：廣告不實

《就業服務法》第5條：
- 招募廣告不實：30萬起跳
- 最高150萬元

**案例：**
- 廣告寫「月休8天」
- 面試說「忙的時候不能休」
- **罰則：30萬-150萬**

---

### 7. 自保之道

#### 雇主（老闆/HR）檢查清單：

- [ ] 年度總休假數是否低於120天？
- [ ] 若低於120天，是否有明確計算並給付加班費？
- [ ] 勞動契約是否有「工資包含加班費」條款？
- [ ] 勞工是否簽名同意？

#### 勞工（求職者）面試問題：

- 月休8天是否包含國定假日？
- 國定假日出勤有加倍薪資嗎？
- 月休8天不符一例一休部分，加班費如何計算？

---

## 📊 計算公式

### 應休假總天數
```
一年應休天數 = (365 ÷ 7) × 2 + 16
             = 104 + 16
             = 120天
```

### 每月平均休假
```
每月平均休假 = 120 ÷ 12 = 10天/月
```

### 月休8天違法差額
```
違法差額 = 10 - 8 = 2天/月
         = 24天/年
```

### 最低加班費計算
```
國定假日加班費 = 時薪196 × 8小時 × 2倍
               = 3,136元/天

休息日加班費 = 時薪196 × 8小時 × 1.33倍（前2小時）
            + 時薪196 × 8小時 × 1.67倍（後6小時）
            = 2,085 + 2,619
            = 4,704元/天
```
