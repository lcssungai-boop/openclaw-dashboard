# 外國人生活管理計劃

**依據：** 就業服務法及相關法規  
**適用對象：** 雇主聘僱外國人

---

## 法規依據

依據雇主聘僱外國人許可及管理辦法第16條規定，雇主應依所聘僱外國人人數，依比率提供生活照顧服務費。

---

## 內容要素

### 一、住宿安排
- 宿舍地點與設施
- 空間標準（每人多少平方公尺）
- 消防安全設備
- 衛生環境

### 二、飲食安排
- 餐廳或炊煮場所
- 食材採購
- 費用分擔

### 三、生活照顧
- 語言翻譯服務
- 緊急聯絡機制
- 文化活動安排

### 四、醫療保健
- 健保就醫資訊
- 緊急就醫流程

### 五、匯款協助
- 銀行開戶協助
- 匯款流程說明

---

## 雇主責任

1. 依規定繳納「就業安定費」
2. 提供安全合法的住宿環境
3. 協助外國人適應在台生活
