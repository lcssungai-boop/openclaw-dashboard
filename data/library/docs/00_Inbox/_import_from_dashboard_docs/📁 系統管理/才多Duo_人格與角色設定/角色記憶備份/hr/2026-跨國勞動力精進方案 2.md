# 2026年 跨國勞動力精進方案

## 實施日期
- 2026年1月1日上路

## 重點內容

### 1. 加薪換移工名額
| 項目 | 內容 |
|------|------|
| 加薪對象 | 事業單位內「月投保薪資等級最低」的全時工作本國勞工 |
| 加薪額度 | 每人月薪資總額調高不少於2,000元 |
| 移工名額 | 每加薪1位本國勞工 → 多1位移工名額 |
| 上限 | 最高不得超過勞保投保人數10% |

### 2. 移工比例提高
- 移工佔總員工人數：由 40% → 45%

### 3. 外國技術人力薪資門檻
- 全面調升（詳細數額待確認）
- 既有聘僱可延續
- 機構須高於基本薪資

### 4. 觀光旅館業
- 雇主自海外引進外國技術人力須符合相關規定
