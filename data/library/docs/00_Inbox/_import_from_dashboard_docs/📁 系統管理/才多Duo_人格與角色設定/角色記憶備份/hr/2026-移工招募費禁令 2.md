# 移工招募費禁令（114年）

**依據：** 就業服務法修正  
**公告單位：** 勞動部  
**實施時間：** 民國114年（2025年）

---

## 重點摘要

### 禁令內容
- 雇主不得要求移工繳納「招募費」
- 禁止私立就業服務機構向移工收取費用
- 違規罰則

### 影響對象
1. 私立就業服務機構（仲介）
2. 雇主
3. 移工

---

## 過渡期

- 實施前已繳納招募費者不退還
- 3年緩衝期讓產業調適

---

## 對才多多影響

### 衝擊
- 招募費收入消失
- 需轉型收費模式

### 建議方向
1. 開發「直聘代辦」服務
2. 強化「後續服務」收費
3. 擴大僑外生業務
