# 授權條款

在本使用條款規範下， 網站名稱   所提供之服務僅限於網站會員使用，包括 **企業會員與求職者會員**。

- 會員不得將帳號用於商業用途
- 不得將帳號分享、出租、出借或轉讓給他人

### **企業會員責任**

- 企業僅可使用求職者資料進行招募
- 不得對求職者造成任何損害或損失
- 網站名稱   不對企業所提供的任何服務結果負責

### **求職者會員責任**

- 求職者應提供 **正確且合法的個人資料**
- 僅可使用企業所提供之資料作為求職用途
- 不得對企業造成任何損害或損失
- 網站名稱   不對求職者與企業間的任何行為負責

### **法律遵循**

所有會員皆應遵守《就業服務法》、《勞動基準法》及其他台灣相關法律規定，任何違法行為皆嚴格禁止。

  網站名稱   僅提供 **企業與求職者之媒合服務**，不負任何法律責任。

若   網站名稱   因任何情況停止或終止服務，會員 **不得提出異議或要求賠償**。

但   網站名稱   應於服務終止前 **至少一個月公告並通知所有會員**。
