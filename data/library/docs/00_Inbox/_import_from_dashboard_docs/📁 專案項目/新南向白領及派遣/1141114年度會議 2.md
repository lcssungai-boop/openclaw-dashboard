**主旨：僑外生招募策略重點建議**

關於僑外生招募事宜，彙整重點建議如下：

- 輪班說明：
    
    僑外生偏好固定班。招募會告知需配合輪班，開始招募後依人數狀況進行回報、調整討論。
    
- 住宿補貼：
    
    建議提供「半年全勤住宿補貼」1750元/月。此補貼應不限廠區住宿，不住廠亦可領取，以提高初期穩定性。
    
- 招募管道：
    
    建議將海外招募評點制資格人員列入招募範圍，此方式有助增加招募人數，海外流程簽由學生自行處理。
    
- 語言激勵：
    
    中文能力建議拉開專業補貼差距。例如 B 級以上、C 級可提供更高補貼，可納入儲備幹部計畫評估。
    
- 條件放寬與合規：
    
    建議開放「英文系」僑外生，可將「特定國籍」的排除招募範圍，有效增加招募人數。
