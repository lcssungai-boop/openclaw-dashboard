# Agent 資訊收集規範

## 目的
建立一致的資訊收集標準，供才多多相關Agent使用。

---

## 一、必記錄項目

### 與工作、法規相關的資訊
• 政府政策、法規新知
• 討論後的重要結論
• 客戶案例分析
• 產業資訊
• 任何與才多多業務相關的資訊

---

## 二、來源優先順序

1. **本地資料庫** - Obsidian（~/Documents/Obsidian Vault/）
2. **iCloud本地** - 備忘錄、提醒事項
3. **網路搜尋** - Firecrawl（取代 WebSearch/WebFetch）
4. **其他** - 如有需要再請示

---

## 三、筆記存放位置

### Obsidian 路徑
```
~/Documents/Obsidian Vault/📁 政策法規/
```

### 筆記命名規範
- YYYY-標題.md
- 例如：2026-僑外生留台工作.md

### 筆記格式
```markdown
---
標籤: #標籤1 #標籤2
建立日期: YYYY-MM-DD
更新日期: YYYY-MM-DD
---

# 標題

## 重點內容
- 項目1
- 項目2

## 資料來源
- 來源說明
```

---

## 四、Telegram 回覆格式

✅ 使用
• 簡單列表（• 或 -）
• 粗體重點
• 📌 emoji 分類

❌ 避免
• 表格（| 對齊會亂掉）
• Markdown 表格
• 複雜排版

---

## 五、網路搜尋工具

### 使用 Firecrawl（不用 WebSearch/WebFetch）
```bash
firecrawl search "關鍵字" --limit 10 --json
```

### 搜尋範例
- 2026年 外國人聘請 加薪條件 勞動部
- 僑外生 居留證 畢業延期居留 2026

---

## 六、筆記更新原則

• 有新資訊即時更新
• 註明更新日期
• 來不及確認的資訊標註「待確認」
• 重要結論用粗體標註

---

## 七、常用關鍵字

### 業務相關
• 移工、僑外生、工讀派遣
• 成大人力、新南向
• 凱塘、小庫哥

### 法規相關
• 最低工資、基本工資
• 退休金、勞退
• 工作許可、居留
