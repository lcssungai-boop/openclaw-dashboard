# 僑外生入職企業告知完整指南

> **建立日期**：2026/02/23
> **資料來源**：勞動部、國稅局
> **標籤**：#僑外生 #評點制 #企業告知 #入職

---

## 一、僱用期限規定

### 評點制僑外生
| 項目 | 規定 |
|------|------|
| 在職期限 | 最長 **3 年** |
| 展延申請 | 期滿前可申請，**無次數限制** |
| 提醒事項 | 建議期滿前 **4 個月** 開始準備展延資料 |

---

## 二、退休金制度

### 制度比較（115年1月1日施行）

| 身份管道 | 適用制度 | 雇主提繳 |
|---------|---------|---------|
| **外國專業人才**（115/1/1後新到職）| **勞退新制** | **6%** |
| 外國專業人才（115/1/1前已在職）| 可選舊制 | 2-15% 或 6% |
| 評點制僑外生 | 勞退舊制 | 2-15% |

### 說明
- 自 115/1/1 起，外國專業人才免取得永久居留即可適用勞退新制
- 選擇舊制者，須於 115/6/30 前書面表明，選擇後不得變更

---

## 三、所得稅預扣（國稅局規定）

### 居留天數與扣繳率

| 居留天數 | 身份 | 薪資水準 | 扣繳率 |
|----------|------|---------|--------|
| **未滿 183 天** | 非居住者 | 月薪 ≧ 41,205 元（基本工資 1.5 倍）| **18%** |
| **未滿 183 天** | 非居住者 | 月薪 < 41,205 元 | **6%** |
| **滿 183 天** | 居住者 | 一般薪資 | 5% / 12% |

### 重點提醒
- 雇主須於給付薪資時即扣繳，並於次月向國稅局申報
- 僑外生新入境，通常適用 6% 或 18% 的非居住者扣繳率

---

## 四、離職與廢聘規定

### 僑外生離職申報
| 項目 | 規定 |
|------|------|
| 廢聘申請 | 離職後 **3 天內** 向勞動部申請廢聘 |
| 應備文件 | 1. 申請書 2. 原聘僱許可函影本 3. 居留證影本 4. 勞僱雙方同意解約證明文件 |

> ⚠️ 注意：「3 天內廢聘」規定主要適用於移工（藍領），僑外生（白領）亦建議從速辦理，避免逾期受罰。

---

## 五、入職檢查清單

- [ ] 居留證（有效期限）
- [ ] 工作許可
- [ ] 勞健保加保生效日
- [ ] 退休金提繳（新制 6% / 舊制 2-15%）
- [ ] 所得稅預扣（6% 或 18%）
- [ ] 3 年期滿展延日（記錄於日曆）
- [ ] 離職 3 天內廢聘（牢記）

---

## 六、企業告知話術

### 退休金
> 「這位僱外生是 115/1/1 後新到職，屬於外國專業人才，直接適用勞退新制 6%，免永久居留。」

### 重要提醒
> 「提醒貴公司：
> - 這位僱外生工作許可最長 **3 年**，請記得期滿前 **4 個月** 申請展延
> - 如有離職，請務必於 **3 天內** 向勞動部申請廢聘
> - 所得稅：居留未滿 183 天須就源扣繳（6% 或 18%，視薪資而定）」
