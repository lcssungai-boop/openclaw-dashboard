### **一般招生**

- [ ]  學校類別－刪除，在學校資訊就有了
- [x]  就學制（下拉選單：短期進修）－都有短期進修的頁面了，為什麼一般招生的就學制度還有這個選項，這跟另一個頁面有衝突
- [ ]  簽證類別－不懂簽證的意義，來台灣讀書不就只有學生簽嗎
＞簽證類別改為「就讀學程」，下拉選單選項如下（跟學校的學程是關聯的，選項新增的話，這邊也會需要同步新增）
    1. 一般學程
    2. 國際產學合作專班
        1. 產學攜手合作僑生專班（3+4）
    3. 1+4 國際專修班
    4. 海外青年技術訓練班
    5. 華語研修班
    6. 新型專班
    7. 雙聯學位學程
    8. 境外專班
    9. 其他 
- [ ]  國籍－預設為不限國籍（唯一值，不得複選）、其他國籍（可複選）
- [ ]  新增欄位－授課語言（選項：中文、英文）
- [ ]  電話－如果是給國外的學生看的，這樣是不是該用國際碼
- [x]  地址備註－不懂寫「備註」的意思，在前台的也面會顯示在哪，目前查看前台只有「地址」欄位，如果是同欄位應該顯示同名稱

- [ ]  錄取公告網站連結－在前台的也面會顯示在哪，目前查看前台沒有類似欄位，只有官網
- [x]  就學迄止日－很少有學校會特地寫這個欄位，何時要畢業這件事學校不會硬性規定
- [x]  建議關於時間的內容要依流程去排序，這樣使用者會比較好順著填，建議排序如下：
報名的日子＞公告錄取名單日＞寄發入學通知＞註冊日期＞就學日
