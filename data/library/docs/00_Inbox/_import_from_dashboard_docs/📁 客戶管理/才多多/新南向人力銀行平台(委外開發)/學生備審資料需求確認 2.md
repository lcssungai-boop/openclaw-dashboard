# 學生備審資料需求確認

ID: 45
類別: 需求討論
狀態: 待討論
負責: PM

目前新南向這邊有反應這邊的界面，不符合實際的需求
這邊實際的需求是，

當學生要來報名學校的時候，會在站內信中一併附上備審資料，
給學校當備審資料，讓校方可以快速的查看這個學生的狀況，來決定要花多久時間來跟學生說明，是一個幫助校方快速過濾學生的機制，。

下面是是一些目前界面有遇到的狀況

1. 語言證明會有不同語言、也有可能完全沒有語言證明，所以不能必填，語言不能只有中文、英文，可能還需要有其他語言。
2. 分數的部分，不同學歷的分數可能有不同的格式，建議採用更有彈性的資料欄位格式。

備審資料頁面的語言證明欄位，

1. 應該要改成有的話可以新增，多筆記錄，然後選擇是什麼語言的語言證明，而不是現在這樣固定就是有中文一個英文一個的證明，。有點類似職缺的說明那樣，可以按加號新增一筆這樣

2. 語言證明不需要必填，因為有的求職者可能完全沒有語言證明。

3. 目前直接送出，會有重複的錯誤提示

[備審資料-錯誤提示重複.jpg](https://drive.google.com/file/d/1yNGXKZVgZS5aDZ6HkaVxjuTUyHhidDjt/view?usp=drive_link)

4. 都選擇填寫後，會有分數必須是整數的提示，但是看不出來是那邊填寫錯誤，

[備審資料-分數必須是整數，但是不知道是那個欄位.jpg](https://drive.google.com/file/d/1mGTAWFZlCLZPgOjkwaXuMEP6sWYvMZdY/view?usp=drive_link)

，這邊應該要像上面一樣是在錯誤的欄位下出現錯誤提示。

5. 目前在後臺只有中文、英文，未來應該也能新增其他語言的語言證明？例如日文....其他的，這邊新南向有提到說因為語言證明的分數，有很多類型，其實可以不用很嚴格的限制是整數，因為有的可能會有小數點。
