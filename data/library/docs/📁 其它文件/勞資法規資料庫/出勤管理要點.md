# 出勤管理要點

## 核心原則
出勤紀錄完整性 = 企業最大防線

## 法規依據
- 勞動基準法
- 勞動事件法 (舉證責任)

## 防範「故意滯留」策略

### 1. 到點驅離
- 下班時間到就請員工離開
- 避免滯留

### 2. 門禁管制
- 非加班時段無法進入
- 無法「假裝加班」

### 3. 加班申請制
- 沒事先申請 = 不算加班

### 4. 監視器佐證
- 畫面證明實際離開時間

## 實務做法
1. 下班時間到 → 廣播/主管提醒「請離開工作區域」
2. 非加班申請不得滯留
3. 隔天核對 → 再與對方確認
4. 每日核對異常紀錄
5. 每月請員工簽認出勤紀錄

## 保存期限
- 打卡紀錄：5 年
- 薪資清冊：5 年
- 出勤紀錄：5 年
