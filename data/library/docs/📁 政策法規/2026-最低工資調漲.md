# 2026年 最低工資調漲 - 中小企業重點須知

## 調整幅度 | 202
| 項目5年 | 2026年 | 漲幅 |
|------|--------|--------|------|
| 月薪 | 28,590元 | 29,500元 | +910元 |
| 時薪 | 190元 | 196元 | +6元 |

## 中小企業老闆要注意的7大關鍵

### 1. 底薪低於基本工資 = 違法
- 月薪低於29,500元 → 違反勞基法
- 時薪低於196元 → 違反勞基法
- **罰則：2萬至100萬元**

### 2. 加班費跟著漲
- 加班費計算基礎改用29,500元
- 前2小時：時薪×1.34
- 後2小時：時薪×1.67
- 注意：是用「總月薪」計算，不是用底薪

### 3. 資遣費增加
- 平均工資受基本工資調漲影響
- 資遣費計算金額隨之增加
- 新制資遣費：每滿1年發給0.5個月

### 4. 勞健保負擔增加
- 勞保級距對應調整
- 公司負擔保費跟著增加

### 5. 發薪日提前（2026年新制）
- 工資發放日提前至「約定日」或「期日」發放
- 不得遲於次月10日

### 6. 保全人員要注意
- 適用84條之1的保全人員
- 正常工時超過每週40小時要另計
- 不能只用基本工資29,500元

### 7. 移工相關
- 外籍移工需提撥舊制退休金
- 最低工資調漲，移工薪資一併調整

## 雇主常見違法行為
1. 底薪暗含加班費（未額外給付）
2. 延長工時未給加班費
3. 投保薪資以高薪低報
4. 未如實申報勞退
